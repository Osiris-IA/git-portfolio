# Portfolio - Projet Git 

Site portfolio personnel développé dans le cadre du projet Git.
Ce projet à pour but de montrer toutes les bonnes pratiques Git et du Git Flow.

## Installation 

- Cloner le repository 

```bash
git clone git@github.com:Osiris-IA/git-portfolio.git
```
- Installer le hook de pre-commit

```bash
cp hooks/pre-commit .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
npm install -g htmlhint
```
